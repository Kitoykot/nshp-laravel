Создать MySQL таблицу в БД, назвать её naruto-shop-laravel или своим именем

в терминале прописать следующее:
composer update

далее переименовать .env.example на .env
в .env изменить данные для подключения к вашей БД

 -- DB_HOST=127.0.0.1 // хост (или своё значение)
 -- DB_PORT=3306 // порт (или своё значение)
 -- DB_DATABASE=naruto-shop-laravel // имя БД (или своё значение, как вы назвали таблицу)
 -- DB_USERNAME=root // логин для доступа в БД (или своё значение)
 -- DB_PASSWORD= // пароль для доступа в БД или своё значение 

в терминале прописать следующее:
php artisan key:generate

теперь прописываем в терминале:
php artisan migrate

в терминале прописать: php artisan storage:link, чтобы загружались и отображались изображения

запускаем сайт через команду php artisan serve или любой другой метод
